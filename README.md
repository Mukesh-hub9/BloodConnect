🩸 BloodConnect – Smart Blood Donation & Emergency Response Platform

BloodConnect is a modern full-stack healthcare web application designed to simplify and speed up the process of blood donation and emergency blood requests. The platform connects donors, patients, hospitals, and emergency responders through a real-time digital system that helps users quickly locate compatible blood donors nearby during critical situations.

The application is built using HTML, CSS, JavaScript, Node.js, Express.js, and SQLite, with an interactive and responsive user interface focused on accessibility, performance, and real-world usability. BloodConnect includes intelligent donor management, emergency request handling, live blood inventory tracking, location-based donor discovery using map integration, and a responsive dashboard for seamless user experience across devices.

Key features of the platform include:
• Donor registration and management
• Blood group-based smart filtering
• Location-aware donor search
• Emergency blood request system
• Interactive OpenStreetMap integration
• Admin dashboard and controls
• AI-powered assistant support
• Dark mode support
• Real-time activity and statistics display
• Offline localStorage fallback system

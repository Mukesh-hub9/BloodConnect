🩸 BloodConnect – Full Stack Blood Donation & Emergency Response Platform

BloodConnect is a modern full-stack healthcare web application designed to simplify and speed up the process of blood donation and emergency blood request management. The platform connects blood donors, hospitals, patients, and emergency responders through a real-time digital system that helps users quickly locate compatible blood donors nearby during critical situations.

This project was developed as a real-world healthcare solution with modern UI/UX principles, responsive design, map integration, emergency handling systems, donor management features, and backend API functionality. The platform demonstrates practical implementation of frontend development, backend architecture, database management, deployment workflows, and real-world problem solving in the healthcare domain.

🎯 Objective (In Simple Words)

The main goal of this project is to create a platform that helps patients and hospitals quickly find blood donors during emergency situations. The system allows users to:

* Register as blood donors
* Search for nearby compatible donors
* Raise emergency blood requests
* Track blood inventory availability
* View donor locations on maps
* Manage donor and request data through admin controls
* Improve emergency response efficiency

This project helped in understanding how real-world healthcare and emergency systems can be built using full-stack web technologies.


🚀 Key Features

🩸 Donor Registration System

Users can register themselves as blood donors by providing:

* Name
* Blood Group
* Phone Number
* Location
* Availability Status

The donor data is stored and managed dynamically through backend APIs and database integration.


🔍 Smart Donor Search

The platform allows users to search donors based on:

* Blood group
* City/location
* Availability status

This helps emergency responders quickly identify compatible donors nearby.


📍Location-Based Donor Discovery

Integrated OpenStreetMap and Leaflet.js to:

* Display donor locations
* Show nearby available donors
* Improve emergency navigation and accessibility


🚨 Emergency Blood Request System

Hospitals or users can create emergency blood requests with:

* Blood group required
* Hospital details
* Urgency level
* Contact information

The requests are displayed dynamically on the dashboard for quick response.


 🏥 Admin Dashboard
